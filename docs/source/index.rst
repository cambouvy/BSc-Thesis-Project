QLKNN-develop
=============
Welcome! This is the documentation manual for the developer tools to
create Neural Networks for the gyrokinetic reduced model QuaLiKiz
`QuaLiKiz <http://qualikiz.com>`_.
QuaLiKiz-pythontools is an open source project under the MIT license.
Please contribute by opening a feature request or issues
`on GitLab <https://gitlab.com/Karel-van-de-Plassche/QLKNN-develop/-/issues>`_
or open a `merge request <https://gitlab.com/Karel-van-de-Plassche/QLKNN-develop/-/merge_requests>`_.
